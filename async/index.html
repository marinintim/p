Анонимные функции

	        return 42;
	      }

Переменной favouriteFunction присвоена анонимная функция. Мы можем присвоить переменной другое значение.

Функции высшего порядка

Коллбэки

Обычные функции, вся соль — в их использовании. Код, который ждёт результатов «заворачивается» в коллбэк.

Отличается от функций тем, что обычно не возвращает значение, а передаёт управление.

В node.js принято, что один коллбэк для всего, первый параметр — ошибка, второй — результат, в jQuery разные коллбэки для ошибок и успеха.

Генераторы

Функции с возможностью прервать выполнение, а затем продолжить в том же контексте.

Генераторы

Изначально придуманы для итераторов (≈ для последовательного обхода структур), но подходят и для наших целей.

	      function* hellogen() {
	        var a = 5
	        yield 'HELLO' // Отдали контроль обратно и передали 'HELLO'
	        yield 'WORLD'
	        return a // a всё ещё доступно, хотя мы дважды покидали функцию
	      }

Генераторы

	      gen = hellogen() // Объект с методом .next()
	      console.log(gen.next()) // { value: 'HELLO', done: false }
	      console.log(gen.next()) // { value: 'WORLD', done: false }
	      console.log(gen.next()) // { value: 5, done: true }

Генераторы

Вместе с подходящей библиотекой, например, co, можно делать так

	      co(function*() {
	        var user = yield getUserById(id)
	        var text = yield fetch('/blacklist')
	        var banned = text.split(' ')
	        if (user.name in banned) return 'you are banned!'
	        return 'welcome!'
	      })
